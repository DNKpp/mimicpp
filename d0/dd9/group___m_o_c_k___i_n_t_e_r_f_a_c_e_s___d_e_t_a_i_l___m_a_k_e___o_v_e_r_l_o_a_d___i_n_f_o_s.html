Base overload, extending the overload info (enclosed by parentheses).

Parameters
retThe return type.
param_type_listThe parameter types.
specsAn optional parameter for categories (e.g. const, noexcept, etc.).
call_conventionAn optional parameter for the utilized call-convention.

Strips all optional parenthesize from the arguments.

Note
No parens will be stripped from ret, because a return type may contain commas (e.g. std::tuple<int, int>). This must be done in a later step.
