Detailed Description

Reporter integrations for various third-party frameworks.

These reporters are specialized implementations, which provide seamless integrations of mimic++ into the desired unit-test framework. Integrations are enabled by simply including the specific header into any source file. The include order doesn't matter.

Note
Including multiple headers of the adapters subdirectory into one executable is possible, but with caveats. It's unspecified which reporter will be active at the program start. So, if you need multiple reporters in one executable, you should explicitly install the desired reporter on a per test case basis.

Reporter for the integration into Catch2.

This reporter enables the integration of mimic++ into Catch2 and prefixes the headers of Catch2 with catch2/.

This reporter installs itself by simply including this header file into any source file of the test executable.
