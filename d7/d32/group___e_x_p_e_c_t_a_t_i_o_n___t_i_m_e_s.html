Detailed Description

Times indicates, how often an expectation must be matched.

During each expectation building users can specify a times policy once. If not specified, that policy defaults to once. If users attempt to specify a times policy more than once for a single expectation, a compile-error will occur.

Times in general have both, a lower and an upper limit. Both limits are treated as inclusive.

Specifies arbitrary limits with compile time arguments.

Template Parameters
minThe lower limit.
maxThe upper limit.
Returns
The newly created policy.
namespace expect = mimicpp::expect;
mimicpp::Mock<void()> mock{};
SCOPED_EXP mock.expect_call()
| expect::times<1u, 3u>(); // between 1 and 3 matches
mock(); // fine
mock(); // fine
mock(); // fine (exhausted)
Note
If the max is omitted, it is set to min.
namespace expect = mimicpp::expect;
mimicpp::Mock<void()> mock{};
SCOPED_EXP mock.expect_call()
| expect::times<2u>(); // equivalent to twice()
mock(); // not enough
mock(); //fine
