Detailed Description

Type-trait, which provides the case-folding algorithm for the char-type, they are specialized for.

This implements the case-folding algorithm for the various character-types. Users should not make any assumptions on what any converter will return, as this is highly implementation specific. No guarantees are made whether the result will be in upper- or lower-case (or something completely different). The only guarantee is, that two inputs, which are required to compare equal after the case-folding process, yield the same result.

See also
https://unicode-org.github.io/icu/userguide/transforms/casemappings.html#case-folding
https://www.unicode.org/Public/UNIDATA/CaseFolding.txt
Note
Users are allowed to add specializations as desired.
